# 📚 Design and Analysis of Algorithms (DAA) – Python Programs

This repository contains Python implementations of common **Design and Analysis of Algorithms (DAA)** programs. These programs are simple, beginner-friendly, and useful for college practicals and lab exams.

## 📋 Algorithms Included

### 🔍 Searching Algorithms

* Linear Search
* Binary Search
* Boyer–Moore String Matching

### 📊 Sorting Algorithms

* Selection Sort
* Quick Sort

### 🌳 Graph Algorithms

* Breadth First Search (BFS)
* Floyd–Warshall Algorithm
* Warshall's Algorithm (Transitive Closure)

### ♟️ Backtracking Algorithms

* N-Queen Problem
* Subset Sum Problem

### 🗼 Recursion

* Tower of Hanoi

### 🌲 Dynamic Programming

* Optimal Binary Search Tree (OBST)

### ➗ Divide and Conquer

* Find Minimum and Maximum

## 🛠️ Requirements

Install Python 3 and the required library:

```bash
pip install matplotlib
```

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/DAA-Python.git
```

Move into the project folder:

```bash
cd DAA-Python
```

Run any program:

```bash
python filename.py
```
